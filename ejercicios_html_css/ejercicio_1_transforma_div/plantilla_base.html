Mi primer artículo sobre HTML5

Publicado: 15 de enero de 2025

HTML5 ha revolucionado la forma en que estructuramos las páginas web. Con la introducción de elementos semánticos como header, nav, article, section y footer, ahora podemos crear documentos web que no solo se ven bien, sino que también tienen significado estructural.

Esto es especialmente importante para la accesibilidad y el SEO. Los lectores de pantalla y los motores de búsqueda pueden entender mejor la estructura de nuestras páginas.

Introducción a CSS y el modelo de caja

Publicado: 20 de enero de 2025

El modelo de caja es fundamental para entender cómo CSS calcula las dimensiones de los elementos. Cada elemento HTML se puede visualizar como una caja rectangular compuesta por cuatro partes: contenido, padding, border y margin.

Comprender este modelo es esencial para crear diseños precisos y predecibles. La propiedad box-sizing nos permite controlar cómo se calculan las dimensiones finales de nuestros elementos.

Formularios accesibles con HTML5

Publicado: 25 de enero de 2025

HTML5 introdujo nuevos tipos de inputs que mejoran enormemente la experiencia del usuario, especialmente en dispositivos móviles. Los tipos como email, tel, date y number no solo proporcionan validación nativa, sino que también activan los teclados apropiados en móviles.

Además, las pseudo-clases CSS como :valid e :invalid nos permiten dar feedback visual inmediato al usuario mientras completa un formulario.
